# Portfolio

A collection of quantitative economics and data science projects spanning financial econometrics, structural modelling, and machine learning.

## Projects

### [ECB Monetary Policy and Eurozone Financial Markets](./ecb-monetary-policy-vecm)
A trivariate VECM analysis examining how ECB unconventional monetary policy — including negative interest rates and quantitative easing — affected Eurozone equity markets and exchange rates over 2015–2018. Implements unit root testing, Johansen cointegration, impulse response functions and a Robinson (1995) long memory test. Built in R and Stata.

### [Structural Entry Models — MLE and Simulation](./structural-entry-models-mle)
Maximum likelihood estimation of the Bresnahan and Reiss (1991) binary entry model, implemented from scratch in MATLAB with analytical gradient and Hessian, cross-validated against Stata. Extends to a Berry (1992) simulation framework incorporating firm heterogeneity. Scored 100%.

### [Brexit and the London Stock Exchange](./brexit-lse-dissertation)
Undergraduate dissertation investigating firm-level heterogeneity in stock market responses to the Brexit referendum. Examines why some firms were affected more than others using econometric methods in Stata.
